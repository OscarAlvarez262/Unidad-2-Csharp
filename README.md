💻 Prácticas C# - Unidad 2

Este repositorio contiene una colección de prácticas y ejercicios en C# desarrollados durante la Unidad 2 de la materia de programación. El objetivo principal es reforzar los conceptos básicos del lenguaje mediante programas de consola que ayudan a comprender mejor la lógica de programación y la estructura de un proyecto en C#.

📂 Contenido del repositorio

El repositorio está organizado principalmente en programas y archivos relacionados con los ejercicios realizados en clase.

• Programas en C#
Contienen los ejercicios desarrollados durante la unidad para practicar lógica de programación y estructuras del lenguaje.

• Clases auxiliares
Archivos que incluyen validaciones y métodos reutilizables utilizados dentro de los programas.

• Archivos de práctica
Ejercicios adicionales utilizados para reforzar los conocimientos vistos durante la clase.

🚀 Temas principales abordados

A lo largo de las prácticas se desarrollan programas que cubren:

• Manejo de variables y tipos de datos

• Entrada y salida de datos en consola

• Uso de condicionales (if, else, switch)

• Uso de ciclos (for, while, do-while)

• Métodos y funciones

• Validación de datos

• Organización básica de programas en C#

🛠️ Requisitos

Para ejecutar los programas necesitas tener instalado:

• .NET SDK

• Visual Studio o Visual Studio Code

• Sistema operativo compatible con .NET
